发表于2023-03-15|更新于2023-03-16|高项

1. 计算机系统的可用性定义（只考选择）

MTTF/(MTTF+MTTR) *100%

MTTF：平均故障时间
MTTR：平均维修时间

2. 投资回报率 ROI （只考选择）

ROI = (总的折现收益 - 总的折现成本) / 总的折现成本

3. 沟通渠道（n 代表干系人的数量）（只考选择）

沟通渠道 = n * (n-1) /2

4. EMV：期望货币价值（只考选择）

EMV = 概率1 *影响1 + 概率2 * 影响2 + 概率3 * 影响3 +... +概率n*影响n

7. 自由时差和总时差

ES：最早开始时间
LS：最迟开始时间
EF：最早结束时间
LF：最迟结束时间
总时差 = LS - ES = LF-EF
自由时差 = min(紧后工作的ES-此活动的EF)

8. PERT 技术

期望时间 T1
悲观时间 T2
乐观时间 T3
最可能时间 T4
贝塔分布：
T1 = （T2+T3+4*T4）/ 6

三角分布：
T1 = (T2+T3+T4)/3

9. 标准差

标准差 = （悲观时间 - 乐观时间）/6


即：

-1& 至 +1& 的面积是 68%

-2& 至 +2& 的面积是 95%

-3& 至 +3& 的面积是 99%

整个图形的面积是 100%，任何一个区域的面积都可计算

10. 挣值分析和预测技术

名词解释：
EV 开始点到检查点已经完成的工作按照计划的价值
PV 开始点到检查点需要完成的工作按照计划的价值
AC 开始点到检查点实际的支出
CV 成本偏差； 如果大于 0，则表示费用节省，加大费用投入；小于 0 则表示费用超支，减少费用投入
SV 进度偏差； 如果大于 0，则表示进度提前，小于 0 则表示进度滞后
CPI 成本绩效指数；如果大于 1，则表示费用节省，加大费用投入；小于 1 则表示费用超支，减少费用投入
SPI 进度绩效指数；如果大于 1，则表示进度提前，小于 1 则表示进度滞后
BAC 完工预算
EAC 完工时成本估算
ETC 完工尚需成本估算
VAC 完工偏差
TCPI 完工尚需绩效指数
CV=EV-AC

SV=EV-PV

CPI=EV/AC

SPI=EV/PV

当偏差看作是非典型的：

ETC = BAC-EV

当偏差看作是代表未来的典型偏差：

ETC=(BAC-EV)/CPI

EAC=AC+ETC 衍化下面两个公式

EAC=AC+BAC-EV 当偏差看作是非典型的
EAC= AC+ (BAC-EV)/CPI 当偏差看作是代表未来的典型偏差
VAC = BAC-EAC

TCPI= (BAC-EV)/(BAC-AC) BAC 维持不变的情况

TCPI= (BAC-EV)/(EAC-AC) 检查点 BAC 发生变更
